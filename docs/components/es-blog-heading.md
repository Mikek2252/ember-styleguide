# Blog Heading

## Regular Post

```handlebars
    {{es-blog-heading
      author='Sara Poster'
      postDate='03-20-2018'
      postUrl='https://emberjs.com/blog'
      postTitle='Pretty Awesome Blog Post'
    }}
  
```

## Guest Post

```handlebars
    {{es-blog-heading
      author='Sara Poster'
      postDate='03-20-2018'
      postUrl='https://emberjs.com/blog'
      postTitle='Pretty Awesome Blog Post'
      isGuestPost=true
    }}
  
```
